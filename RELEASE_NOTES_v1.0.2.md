# System Vitals v1.0.2

**전체화면 고정 시 작업표시줄이 덮이지 않던 문제 수정.**

---

## 🐛 수정

**전체화면에서 작업표시줄이 보이던 버그**
- 전체화면(구 "키오스크 잠금")을 켜면 대시보드가 작업표시줄 위까지 덮어야 하는데, v1.0.1 이후 하단 작업표시줄 바가 그대로 보이던 문제를 고쳤습니다.
- 원인: v1.0.1 에서 작업표시줄 **버튼**을 없애며 창에 붙은 툴윈도우 스타일 탓에, Windows 가 이 창을 "전체화면 앱"으로 인식하지 않아 작업표시줄을 자동으로 숨기지 않았습니다.
- 해결: 전체화면일 때 창을 작업표시줄보다 위(z-order)로 올려 직접 덮습니다. 작업표시줄 버튼은 여전히 표시되지 않습니다(백그라운드 실행 유지).

## ✏️ 변경

- 설정 → 디스플레이의 **"키오스크 잠금" → "전체화면"** 으로 이름 변경. 의미도 명확하게: *작업표시줄까지 덮어 이 모니터에 고정.*

## 📦 다운로드 & 실행

- **`System Vitals 1.0.2.exe`** *(포터블)* — 설치 없이 바로 실행
- **`System Vitals Setup 1.0.2.exe`** *(설치 프로그램)*

> 서명되지 않은 빌드라 SmartScreen 경고가 뜰 수 있습니다: **추가 정보 → 실행**.
> v1.0.1 사용자는 앱 내 **설정 → 일반 → 업데이트 확인** 에서 이 버전을 안내받을 수 있습니다.
