# System Vitals v1.0.3

**앱 안에서 바로 다운로드하고 설치까지 — 원클릭 업데이트.**

---

## ✨ 개선

**인앱 업데이트 (다운로드 + 설치)**
- 이전에는 새 버전 알림 시 릴리즈 페이지로 안내만 했지만, 이제 **설정 → 일반 → "다운로드 후 설치"** 버튼 하나로 처리됩니다.
- **설치본**: 앱 안에서 진행률과 함께 설치 프로그램을 받아 **자동 실행** → 클릭 몇 번으로 버전업(설치 후 자동 재시작).
- **포터블**: 새 버전을 받아 폴더를 열어 드립니다(실행 중인 파일은 스스로 교체할 수 없어, 새 파일로 교체하면 됩니다).
- 실행 형태(설치본/포터블)를 자동 감지해 알맞은 파일을 받습니다.
- 실패하거나 직접 받고 싶을 땐 **"릴리즈 페이지에서 직접 받기"** 링크도 그대로 제공합니다.

> ⚠️ 서명되지 않은 빌드라, 설치 프로그램 실행 시 **SmartScreen 경고**가 한 번 뜰 수 있습니다: **추가 정보 → 실행**. (완전한 무경고 자동 설치는 코드 서명이 있어야 가능합니다.)

## 📦 다운로드 & 실행

- **`System Vitals 1.0.3.exe`** *(포터블)*
- **`System Vitals Setup 1.0.3.exe`** *(설치 프로그램)*

> v1.0.1 이상 사용자는 앱 내 업데이트로, v1.0.2 사용자는 이 버전부터 원클릭 업데이트를 이용할 수 있습니다.
