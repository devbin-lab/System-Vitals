# System Vitals v1.0.1

**작업표시줄에서 사라지는 백그라운드 실행 + 인앱 업데이트 확인.**

---

## 🔧 변경 사항

**작업표시줄 숨김 (백그라운드 실행)**
- 실행해도 **작업표시줄에 버튼이 생기지 않습니다**. 대시보드는 지정한 서브모니터에 그대로 뜨고, 앱은 조용히 백그라운드로 동작합니다.
- 제어는 **시스템 트레이 아이콘**(좌클릭 = 표시, 우클릭 = 메뉴)과 화면 내 **설정(⚙)** 창으로.
- Windows에서 전체화면 전환 · 모니터 재연결 · 키오스크 토글 후 작업표시줄 버튼이 되살아나는 문제를 예방하는 가드 포함.

**인앱 업데이트 확인**
- 설정 → **일반** 탭에서 GitHub 최신 릴리즈를 확인합니다. 새 버전이 있으면 상태 표시 + 릴리즈 노트 요약 + **다운로드 페이지 열기** 버튼, 그리고 설정 기어(⚙)에 알림 점이 붙습니다.
- 시작 후 자동으로 한 번 확인하고, "업데이트 확인" 버튼으로 언제든 다시 확인할 수 있습니다.
- 서명되지 않은 포터블/설치본이라 자동 설치는 하지 않고, **새 버전 알림 + 다운로드 안내**까지만 합니다. 다운로드는 브라우저에서 직접 받습니다.

> ℹ️ v1.0.0 에는 이 기능이 없어 자동으로 알림받지 못합니다. v1.0.1 은 아래에서 **수동으로** 받으세요. 이후 버전부터는 앱이 새 릴리즈를 자동으로 알려줍니다.

## 📦 다운로드 & 실행

- **`System Vitals 1.0.1.exe`** *(포터블)* — 설치 없이 바로 실행
- **`System Vitals Setup 1.0.1.exe`** *(설치 프로그램)* — 설치 경로 선택 가능(NSIS)

> 서명되지 않은 빌드라 SmartScreen 경고가 뜰 수 있습니다: **추가 정보 → 실행**.
