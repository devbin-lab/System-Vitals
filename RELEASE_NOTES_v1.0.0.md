# System Vitals v1.0.0

**전용 서브모니터를 위한 실시간 PC 하드웨어 모니터링 대시보드** — 첫 정식 릴리즈.

CPU · GPU · 메모리 · 전력 · 스토리지 · 팬 · 네트워크를 한 화면에서. 설치 없이 실측하고, 원하는 서브모니터에만 띄웁니다.

---

## ✨ 주요 기능

**실시간 실측**
- `nvidia-smi`(GPU) + Windows 성능 카운터·WMI(CPU·RAM·네트워크·디스크·부품 신원)로 **설치 없이** 실측
- *(선택)* LibreHardwareMonitor 연동 시 **CPU 온도·전력, 메인보드 팬 RPM, 디스크 온도**까지
- 읽을 수 없는 값은 가짜로 채우지 않고 그대로 표기

**커스텀 위젯 시스템**
- 안드로이드 홈 위젯처럼 격자에 **배치·크기 조절** (편집 모드에서 모서리 드래그)
- 원그래프 게이지 · CPU/GPU 종합 패널 · 전력/네트워크 꺾은선 그래프 · 코어·팬·스토리지 리스트 등 **20여 종**, 카테고리별 팔레트
- 위젯 크기에 맞춰 폰트·게이지·열 수가 **자동 최적화**(container query)
- **격자 밀도 조절** — 해상도가 달라도 같은 배치를 재현
- **CPU/GPU 통합 게이지** — 링 길이=사용률, 링 색=온도 심각도, 중앙=온도

**UI / 테마**
- **2-pane 설정 창**(좌 네비 + 우 콘텐츠): 모양 · 레이아웃 · 디스플레이 · 일반
- 라이트 / 다크 / 시스템 테마, UI 배율 조절
- 프리미엄·미니멀 디자인, FHD/QHD 자동 정돈, 터치 지원

**서브모니터 & 성능**
- 원하는 모니터에만 전체화면/키오스크 고정
- **조건부 자동 실행** — 부팅 시 자동 시작 + **지정한 모니터가 연결될 때만** 표시(없으면 트레이에서 대기, 연결 시 자동 표시). 트레이 아이콘으로 표시·설정·종료
- GPU 가속 off + 상시 애니메이션 정리로 **CPU/RAM 사용 최소화**

## 📦 다운로드 & 실행

- **`System Vitals 1.0.0.exe`** *(포터블)* — 설치 없이 바로 실행
- **`System Vitals Setup 1.0.0.exe`** *(설치 프로그램)* — 설치 경로 선택 가능(NSIS)

> 실행하면 지정한 서브모니터에 대시보드가 전체화면으로 뜹니다. 처음엔 커서가 있는 모니터에 뜨며, 설정(⚙)에서 원하는 모니터·키오스크를 지정할 수 있습니다.

## ⚙️ 요구 사항

- Windows 10 / 11
- *(선택)* NVIDIA 드라이버 — GPU 실측
- *(선택)* [LibreHardwareMonitor](https://github.com/LibreHardwareMonitor/LibreHardwareMonitor)(관리자) — CPU 온도/전력·메인보드 팬·디스크 온도. 대시보드 안내 배너에서 자동 설치 가능
- 실측 수집기는 Python 3 를 사용하지만 **Windows 기본 도구만** 쓰므로 별도 설치는 필요 없습니다

## 📝 참고

- CPU 온도·전력, 메인보드 팬, 디스크 온도가 `—` 로 뜨면 LibreHardwareMonitor 를 켜세요(Remote Web Server :8085).
- 위젯 배치·테마·UI 배율은 자동 저장됩니다.
